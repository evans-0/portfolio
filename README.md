## 🏗️ Built With

Pure **HTML, CSS, and vanilla JavaScript** — zero frameworks, zero dependencies. Everything lives in a single `index.html` file.

---

## ✨ Features

- **Responsive design** — fully mobile-friendly with a hamburger navigation menu
- **Scroll reveal animations** — sections fade in as you scroll using `IntersectionObserver`
- **Active nav link tracking** — highlights the current section as you scroll
- **Morphing avatar** — CSS `border-radius` keyframe animation
- **Glassmorphism navbar** — `backdrop-filter: blur()` with scroll shadow
- **Hosted on GitHub Pages** — zero build step, deploys automatically from `main`

## 🚀 Running Locally

No setup needed — just open the file:

```bash

cd portfolio
open index.html        # macOS
xdg-open index.html    # Linux
```

Or just visit the **[live site](https://evans-0.github.io/portfolio/)**.
